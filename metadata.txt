about=Multi Project Canvas allows you to work with multiple QGIS projects at the same time without opening separate QGIS instances. Each project is fully isolated with its own layers, extent, CRS, and bookmarks. Switch between projects with a single click while keeping all your work organized in one place.

    Key Features:
    - Multiple project management in a single QGIS session
    - Live map thumbnails for visual project identification
    - Per-project bookmarks with collapsible panel
    - Navigation history (back/forward) for map extents
    - Extent synchronization across all projects
    - Global search for layers and bookmarks
    - Drag & drop project reordering
    - Workspace save/load to preserve sessions
    - Full i18n support (English, Italian)

    Shortcuts:
    - Ctrl+Shift+P: Toggle panel
    - Ctrl+T: New project
    - Ctrl+Shift+O: Open project

tracker=https://github.com/fgianoli/multy_project_canvas/issues
